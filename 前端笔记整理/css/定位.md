## 定位(position)

position属性的常用值

| static   | 自动定位（默认定位方式）                                     |
| -------- | ------------------------------------------------------------ |
| relative | 相对定位，相对于其原文档流的位置进行定位(原本的位置还占着)   |
| absolute | 绝对定位，相对于其上一个已经定位的父元素进行定位(不占原本的位置)(子绝父相) |
| fixed    | 固定定位，相对于浏览器窗口进行定位                           |

 与top  bottom  left  right 搭配使用

**元素 添加了 绝对定位或者 固定定位后 浮动后，直接变为行内块模式，直接给高度和宽度就可以了**
